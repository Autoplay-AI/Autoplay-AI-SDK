# Support

Thanks for using `autoplay-sdk`.

## Where to get help

- `GitHub Issues` for reproducible bugs and feature requests
- `Discord` for setup help, troubleshooting questions, and integration guidance
- `Security issues` must be reported privately via the process in `SECURITY.md`

## Before opening an issue

Please include:

- Python version and operating system
- `autoplay-sdk` version
- Minimal reproduction steps
- Expected behavior vs actual behavior
- Relevant logs or stack traces (redact secrets)

## Response expectations

- We aim to acknowledge new support requests within 3 business days
- Severity and impact determine prioritization and fix timelines
- Security reports follow the stricter SLA documented in `SECURITY.md`
