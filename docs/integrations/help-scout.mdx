<Info>
  **Dedicated helpers for Help Scout are coming soon.** That is **not a blocker**: you can **still stream events to your connector and chatbot (or delivery) endpoints today** using the core SDK and event pipeline. When they ship, these helpers will be **optional extras** — not required for implementation.
</Info>

---

We plan first-party helpers so Help Scout conversations can show the same structured session context as other channels. Until then, use [`ConnectorClient`](/sdk/sync-client) / [`AsyncConnectorClient`](/sdk/async-client) and your own delivery path.
