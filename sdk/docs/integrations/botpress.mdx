<Info>
  **Dedicated helpers for Botpress are coming soon.** That is **not a blocker**: you can **still stream events to your connector and chatbot (or delivery) endpoints today** using the core SDK and event pipeline. See the [Botpress tutorial](/recipes/botpress) for a full end-to-end pattern. When first-party helpers ship, they will be **optional extras** — not required for implementation.
</Info>

---

We plan first-party helpers so Botpress workflows can consume Autoplay session context with less glue code. Until then, use the standard client and push payloads into your own integration layer.
