<Info>
  **Dedicated helpers for Diffy are coming soon.** That is **not a blocker**: you can **still stream events to your connector and chatbot (or delivery) endpoints today** using the core SDK and event pipeline. When they ship, these helpers will be **optional extras** — not required for implementation.
</Info>

---

We plan first-party helpers so Diffy workflows can consume Autoplay session context with less glue code. Until then, use the standard client and push payloads into your own integration layer.
