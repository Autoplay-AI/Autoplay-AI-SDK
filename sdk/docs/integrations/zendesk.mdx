<Info>
  **Dedicated helpers for Zendesk are coming soon.** That is **not a blocker**: you can **still stream events to your connector and chatbot (or delivery) endpoints today** using the core SDK and event pipeline. When they ship, these helpers will be **optional extras** — not required for implementation.
</Info>

---

We plan first-party helpers so you can align Zendesk messaging with the same real-time session context Autoplay already delivers elsewhere. Until then, wire [`ConnectorClient`](/sdk/sync-client) or [`AsyncConnectorClient`](/sdk/async-client) and your chatbot path as you do for any custom destination.
